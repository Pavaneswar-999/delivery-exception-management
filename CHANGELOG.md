# Changelog

## 1.0.0 — 2026-07-15

Initial frontend version for the GreyAtom Logistics Delivery Exception Management assignment.

### Added

- Delivery exception form with required-field validation
- Dynamic operations queue with issue-type and status filters
- Resolve and delete row actions
- Session counters for tracked, open, and resolved exceptions
- Priority and status visual states
- Responsive HTML and CSS layout
- Plain JavaScript DOM manipulation with in-memory session state

### Scope

This version intentionally excludes backend storage, authentication, APIs, persistence, and multi-user synchronization in line with the assignment brief.
